Layout Editing

  • Drag any node to reposition it
  • Double-click a node to rotate it 90°
  • Hover on a node edge to reveal the resize handle
  • Drag the resize handle to change node length

Orientation

  • 0° — inputs left, outputs right
  • 90° — inputs top, outputs bottom
  • 180° — inputs right, outputs left (flipped)
  • 270° — inputs bottom, outputs top
